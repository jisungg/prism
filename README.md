# Prism

Prism is a chess collaboration platform for game review, move explanation, and shared chess study. Instead of stopping at "the engine says this is best", Prism is being built to help strong players, coaches, and curious learners explain why a move works and discuss the ideas behind it together.

## Product Direction

Traditional chess tools optimize for engine accuracy. Prism is aimed at turning engine analysis into human understanding by pairing reviews with discussion, communities, and player-owned game spaces.

Core goals:

- import and parse chess games
- orient each game around the uploading player's side
- generate engine-backed game reviews
- let posters flag one or many confusing moves for community help
- surface generally accepted explanations for difficult engine ideas
- let communities discuss key positions and explain difficult moves
- support private or public chess groups for coaching, prep, and study
