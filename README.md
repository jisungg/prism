# Prism

Prism is a chess analysis platform focused on human behavior, opponent preparation, and exploitability. Instead of only answering "what is the best move in this position?", Prism is being built to answer "what move is most likely to beat this particular player?"

## Product Direction

Traditional chess tools optimize for engine accuracy. Prism is aimed at practical decision-making against real opponents by combining game history, behavioral patterns, and engine-assisted analysis.

Core goals:

- import and parse chess games
- build behavioral models of players
- detect recurring mistakes and style patterns
- generate opponent preparation reports
- suggest practical strategies against specific players
